Sistemas operando actualmente

Plataforma empresarial para la gestión integral del personal y procesos operativos

SCP

Human Capital Management (HCM) Platform

01
Problema

La operación dependía de procesos manuales y herramientas dispersas para administrar personal, horas extra, permisos y accesos, dificultando la trazabilidad, auditoría y generación de información para la toma de decisiones.

Solución

Diseño y desarrollo una plataforma web modular para la administración de personal, implementando autenticación segura, control de accesos por roles, auditoría de operaciones, APIs REST y una arquitectura escalable orientada al crecimiento del sistema.

Tecnologías
Python SQL Server FastAPI JWT Microsoft ENTRA ID
Resultados

Centralización de la información del personal en una única plataforma, mejora en la trazabilidad de procesos, fortalecimiento de la seguridad mediante autenticación y control de accesos, además de una arquitectura preparada para incorporar nuevos módulos.

Sistema de control para la gestión e inventario de equipos móviles

CIEM

Control de Inventario de Equipos Móviles

02
Problema

La administración de montacargas, tractores y otros equipos móviles se realizaba mediante procesos manuales, dificultando conocer su disponibilidad, ubicación, estado operativo y asignación a las diferentes áreas de trabajo.

Solución

Diseño y desarrollo una plataforma web para centralizar el inventario y administración de equipos móviles, permitiendo gestionar su información, clasificación, ubicación, estado y trazabilidad mediante una arquitectura modular basada en ASP.NET Core, Vue y SQL Server.

Tecnologías
ASP.NET Core SQL Server Entity Framework Core Vue.js REST API Git
Resultados

Se centralizó la información de los equipos móviles en una única plataforma, facilitando su consulta, administración y futura integración con nuevos procesos operativos y de mantenimiento.

Sistema de monitoreo y trazabilidad logística en tiempo real

TCC - Truck Control Center

Yard Management System

02
Problema

El seguimiento de las unidades de transporte dentro de las instalaciones se realizaba de forma manual en mayoria, dificultando conocer los tiempos de permanencia, ubicación y estado de cada operación logística.

Solución

Desarrollo de un sistema web para el monitoreo de estadías de transporte en tiempo real, permitiendo registrar eventos operativos, visualizar el estado de las unidades y mejorar la trazabilidad de los procesos mediante una arquitectura basada en ASP.NET Core, Vue y SQL Server.

Tecnologías
ASP.NET Core SQL Server Entity Framework Core Vue.js REST API Git
Resultados

Se mejoró la visibilidad de las operaciones logísticas mediante el seguimiento de las unidades dentro del patio, proporcionando información en tiempo real para apoyar la toma de decisiones y optimizar el control operativo.
